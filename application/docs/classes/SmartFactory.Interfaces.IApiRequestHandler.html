registerApiRequestHandler()

registerApiRequestHandler(string  $api_request, callable  $handler) : boolean

Registeres a handler for an API request.

Parameters

string $api_request

The API request name.
callable $handler

The name or definition of the handler function. The signature of this function is:

function (IApiRequestHandler $handler, string $api_request) : boolean;

  • $handler - the current instance of the handler object.

  • $api_request - the API request name.

  • The handler function should return true, if the request has been successfully handled. Otherwise it should return false to signilize the handler that an error occured.

Returns

boolean —

Returns true if the handler has been successfully registered, otherwise false.

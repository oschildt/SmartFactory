Method: registerApiRequestHandler

public registerApiRequestHandler(string &$api_request, callable &$handler) : boolean
Registers a handler function for an API request call.

Parameters

$api_request

Type: string

The target API request.

$handler

Type: callable

The name or definition of the handler function. The signature of this function is:

function (IApiRequestHandler $handler, string $api_request) : boolean;

  • $handler - the reference to the current handler object.

  • $api_request - the name of the API request for what it was called.

  • The handler function should return true if the request has been successfully handled, otherwise false.

Returns

Type: boolean

Returns true if the registration was successfull, otherwise false.

Method: registerApiRequestHandler

public registerApiRequestHandler(string $api_request, string $handler_class_name) : boolean
Registers a handler action for an API request call.

Parameters

$api_request

Type: string

The target API request.

$handler_class_name

Type: string

The name of the class for handling this API request.

Important! It should be a name of the class, neither the class instance nor the class object. It is done to prevent situation that a wrong registration of a handler breaks the handling of all requests.

The class instantiating and class loading occurs only if this API request comes.

Returns

Type: boolean

Returns true if the registration was successfull, otherwise false.

Throws

\Exception

It might throw the following exceptions in the case of any errors:

  • if the request name is not specified.
  • if the request already has a handler.

Method: registerPreProcessHandler

public registerPreProcessHandler(string $handler_class_name) : boolean
Registers a handler action that will be executed before any other handlers.

Parameters

$handler_class_name

Type: string

The name of the class for handling this API request.

Important! It should be a name of the class, neither the class instance nor the class object. It is done to prevent situation that a wrong registration of a handler breaks the handling of all requests.

The class instantiating and class loading occurs only if this API request comes.

This is usefult for turning on the maintenance mode. The handler may check a a setting value and return the maintenance json for all requests.

Returns

Type: boolean

Returns true if the registration was successfull, otherwise false.

Throws

\Exception

It might throw the following exceptions in the case of any errors:

  • if the request already has a handler.

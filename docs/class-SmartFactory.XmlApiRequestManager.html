Method: parseXML

abstract protected parseXML(string &$api_request, \DOMDocument &$xmldoc) : void
Parses the incoming XML data of the API request.

Details

This method should be implemented by the user. This method is called every time a new API request comes. The user should parse the incoming data. The result of parsing should be the detected API request name and the DOM document.

Parameters

$api_request

The detected DOM document.

Method: registerApiRequestHandler

public registerApiRequestHandler(string $api_request, string $handler_class_name) : boolean
Registers a handler function for an API request call.

Parameters

$api_request

Type: string

The target API request.

$handler_class_name

Type: string

The name of the class for handling this API request.

Important! It should be a name of the class, mneither the class instance nor the class object. It is done to prevent situation that a wrong registration of a handler breaks the handling of all requests.

The class instantiating and class loading occurs only if this API request comes.

Returns

Type: boolean

Returns true if the registration was successfull, otherwise false.

Throws

\Exception

It might throw the following exceptions in the case of any errors:

  • if the request name is not specified.
  • if the request already has a handler.

Name Pass type Value type Default value Description
$message by value string

$error_code by value string

Error code.

Since the error texts can be localized, the unique code of the error might be important fo using in comparison.
$error_type by value string

Error type.

The type of the error might be useful for decision how to show the error on the client. If it is a user error, the full error texts should be shown. If it is a programming error, the detailed text should be shown only in the debug mode to prevent that the hackers get sensible information about the system.

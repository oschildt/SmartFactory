Function: timestamp

timestamp(string &$time_string, string &$format) : integer|string
Converts the string representing the date/time in the specified format to the timestap.

Parameters

$time_string

Type: string

The date/time string.

$format

Type: string

The date format of the date/time string in the PHP systax for the date formats, e.g. "Y.m.d H:i:s".

Returns

Type: integer|string

If the date/time string could be converted, the timestamp is returned, otherwise the string "error" is returned.
